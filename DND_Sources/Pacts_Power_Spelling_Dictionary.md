# Pacts & Power — Spelling Dictionary & Auto-Correction Reference

> **Purpose:** Canonical spelling reference for the Pacts & Power D&D campaign. Use this document to automatically correct speech-to-text transcription errors in session transcripts stored in the vault.
>
> **How to use:** When processing a raw transcript, scan for any term in the "Common Transcription Errors" column and replace it with the corresponding "Canonical Spelling." Do not correct terms that are not listed here. Preserve all other transcript text verbatim.
>
> **Source authority:** Campaign source files > Pacts & Power Spelling Guide > Ravnica Spelling Guide > Official MTG/WotC sources > D&D 5E sources. DM rulings (Chris) override all other sources.
>
> **Last updated:** 2026-06-01 (added finale terms: God-Eternals, Orogalan, Aphanexa, HellSmasher)

## 11. AUTO-CORRECTION RULES

1. **Only correct terms listed here.** Do not invent corrections.
2. **Preserve verbatim dialogue.** Corrections are proper nouns only.
3. **Context matters.** "Ren" → "Rin" only as standalone name, not inside words.
4. **"Raw" → "Ral"** only when addressing Ral Zarek in dialogue.
5. **"Boris" → "Boros"** only when referring to the guild.
6. **Leather Daddy Ball vs Game Boy:** Use session date. Pre-merge = Ball. Post-merge = Game Boy.
7. **"Selena" vs "Salena":** Standardize to "Selena" unless contradicted.
8. **"Zaddy Daddy":** Intentional nickname. PRESERVE IT.
9. **Ghost Mind spirits (Eleni, Gavril, Vula, Sarin, The Whisper Mask, The Veil):** Leave as-is.
10. **Hikara:** Confirmed canonical. Do not correct.
11. **Orphea vs Orphie:** Both correct. Only correct "Orphiea"/"Orphieus" → "Orphea."
12. **"Mardu":** Do NOT correct to "Bontu." "Mardu" is a canonical Tarkir clan (Mardu Horde). The God-Eternal was only loosely called "Mardu" in PP_20 dialogue; handle that contextually in notes, not by global rule.
13. **Orogalan nicknames:** "Urogalan," "Roguelin," "Ogrelin," "Oglin" are intentional in-jokes for the merged Ogre. PRESERVE. Only correct the clear mis-hears (Ogorgalon, Arogalin, Arugalon, Orogalon) → "Orogalan."
