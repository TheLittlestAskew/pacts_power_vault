## MCP Safety Rules

1. Do not search the full vault.
2. Do not recursively list all files.
3. Do not scan all tags.
4. Do not open media folders.
5. Do not open raw transcripts unless Taylor provides the exact file path.
6. Do not open DND source/reference folders unless Taylor specifically names the file.
7. Prefer exact file paths over search.
8. Prefer section-level edits over full-file rewrites.
9. Append proposed changes to `Tracker Patch Queue.md` before editing canon tracker files.
10. If a needed file is not listed here, ask Taylor for the exact path.
