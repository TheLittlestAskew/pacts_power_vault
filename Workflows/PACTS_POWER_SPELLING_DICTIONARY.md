# Pacts & Power — Spelling Dictionary & Auto-Correction Reference

> **Purpose:** Canonical spelling reference for the Pacts & Power D&D campaign. Use this document to automatically correct speech-to-text transcription errors in session transcripts stored in the vault.
>
> **How to use:** When processing a raw transcript, scan for any term in the "Common Transcription Errors" column and replace it with the corresponding "Canonical Spelling." Do not correct terms that are not listed here. Preserve all other transcript text verbatim.
>
> **Source authority:** Campaign source files > Pacts & Power Spelling Guide > Ravnica Spelling Guide > Official MTG/WotC sources > D&D 5E sources. DM rulings (Chris) override all other sources.
>
> **Last updated:** 2026-05-16

## 11. AUTO-CORRECTION RULES

When applying this dictionary to raw transcripts, follow these rules:

1. **Only correct terms listed in this document.** Do not invent corrections.
2. **Preserve verbatim dialogue.** Corrections apply to proper nouns only, not grammar, tone, or speech patterns.
3. **Context matters.** "Ren" should only be corrected to "Rin" when it's clearly a standalone name reference, not part of words like "aren't," "children," "different," "rent," etc.
4. **"Raw" → "Ral"** only when the speaker is clearly addressing Ral Zarek in dialogue. Do not correct "raw" as an adjective.
5. **"Boris" → "Boros"** only when clearly referring to the guild, not when used as a standalone word in other contexts.
6. **"Leather Daddy Ball" vs "Leather Daddy Game Boy":** Use session date to determine. Pre-merge sessions use "Ball." Post-merge sessions use "Game Boy." When in doubt, preserve transcript wording and flag for review.
7. **"Selena" vs "Salena":** Campaign notes use "Selena." Some transcripts use "Salena." Standardize to "Selena" in corrected output unless a specific source contradicts.
8. **"Zaddy Daddy":** This is an intentional table nickname for Szadek. PRESERVE IT. Do not correct to Szadek.
9. **Ghost Mind spirit names (Eleni, Gavril, Vula, Sarin, The Whisper Mask, The Veil):** These are campaign-specific NPC names within the Ghost Mind. Leave as-is unless a correction is explicitly requested by the user.
10. **Hikara:** Confirmed canonical campaign spelling. Do not correct.
11. **Orphea vs Orphie:** "Orphea" is the full canonical name. "Orphie" is the nickname. Both are correct. Do NOT correct Orphea → Orphie. Only correct misspellings like "Orphiea" or "Orphieus" → "Orphea."
