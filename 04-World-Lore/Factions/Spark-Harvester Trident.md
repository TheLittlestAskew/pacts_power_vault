# Spark-Harvester Trident

The weapon used with the **[[Elder Spell]]** to harvest planeswalker sparks. Named by [[Chandra Nalaar|Chandra]] in [[PP_03_081025_Thats_Why_No_One_Will_Remember_Your_Name|Transcript 03]].

## Description & Function
- A trident (also described earlier as a spark-extracting **bident/stave**) carried by the [[Eternals]]; the party noticed in earlier sessions that Eternals "hang back" with these tools to avoid being hit.
- Cast the [[Elder Spell]], then use the trident to **pull a planeswalker's spark** (after death). The harvested sparks flow to [[Bolas's Citadel]] for [[Nicol Bolas]] to absorb.

> ⚠ **Terminology note:** Transcript 03 calls it a **trident** (Chandra's exact words). Earlier sessions/notes describe the Eternals' tools as **bidents/staves** (transcribed "Biden"). Likely the same class of weapon; flagged.
