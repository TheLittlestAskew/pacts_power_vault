# Blistercoils

A Ravnican district/neighborhood — the home of the halfling girl **[[Janna]]**, who calls it the **"Blister Coils."**

> Spelling unconfirmed (raw transcript); canonical page form **Blistercoils** (alias **Blister Coils**).

## Description

Little is established on-screen beyond it being [[Janna]]'s home turf. Her refusal to return there (preferring to tag along with the party) leaves the district's character largely undescribed so far.
